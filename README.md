# 🍽️ Taste of Odisha - Data Analytics Dashboard

## 📊 Overview
This project is a web-based dashboard that analyzes restaurant sales data using a CSV file.

## Live Website
https://swatipanda1105.github.io/Test-of-Odisha-analytics-dashboard/

## 🚀 Features
- Upload CSV file
- Sales trend visualization
- Dish performance analysis
- Revenue distribution
- Automatic insights generation

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## 📂 How to Use
1. Open the website
2. Upload a CSV file
3. View analytics instantly

## 📈 CSV Format
date,dish,amount
2026-03-01,Rasgola,500

## 🎯 Purpose
Built as a beginner data analytics project for learning and portfolio.
